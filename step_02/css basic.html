선택자 : css를 적용시킬 body에 있는 Eliment 선택(지정)

  1. Eliment 선택자(div, p, h1...) :직접 태그명을 입력해서 선택
  2. 자손 선택자(div a{}) :특정태그 하위에 존재하는 태그 선택
  3. 자식 선택자(div>ul>a{}) :하나의 태그 바로 하위(자식요소)를 선택
  4. id, class 선택자 : #, . 등 표기하여 Body에 작성된 이름 선택
  5. 속성 선택자(a[href="#"]{}): 지정된 태크의 속성값이 일치하는 것을 찾아 선택
  6. 가상 선택자(a:hover{}, a:active{}) : 실제로 존재하는 태그. 이름이 아닌 가상으로 표현 선택
